# Vue 3 + Vite

INSTAL AND CONFIGURE CONTAINER DOCKER

1--- Rode o comando :
        npm create vite@latest meu-projeto-vue <-(nome da pasta)

        escolha na instalação :
        Vue
        JavaScript

        Use roldown-vite = NO

        Instalar com npm e iniciar agora? = yes

2--- crtl + C

3--- Copie os arquivos docker-compose-yml e Dockerfile para dentro do seu projeto 

4--- docker compose up --build (para subir o container)



docker compose down --volumes (fechar o container)
